Atomic write — crash safety
1. json.dump() → state.json.tmp
2. shutil.move(state.json.tmp, state.json) # atomic on POSIX
pointer advances per-row inside the loop, not per-batch
crash on row N: restart picks up from row N, not row 0
reset: python3 transfer.py --reset (CLI only — not exposed in UI)
